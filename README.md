# Lista de Tareas

Este proyecto consiste en una aplicación web para la gestión de tareas, desarrollada con HTML, CSS y JavaScript. Permite al usuario crear, organizar y administrar tareas de manera sencilla, incorporando funciones adicionales como filtrado, edición, fechas límite y cambio de tema.

El proyecto fue realizado como práctica para reforzar el manejo del DOM, el uso de almacenamiento local y la mejora de la experiencia de usuario mediante animaciones y estilos personalizados.

---

## Funcionalidades principales

- Agregar nuevas tareas mediante un formulario.
- Editar el texto de una tarea existente directamente desde la lista.
- Marcar tareas como completadas o pendientes.
- Eliminar tareas con una animación de salida.
- Asignar una fecha límite a cada tarea.
- Identificar visualmente tareas próximas a vencer o vencidas.
- Clasificar tareas por categorías:
  - Personal
  - Trabajo
  - Escuela
  - Otro
- Filtrar tareas por estado (todas, pendientes o completadas).
- Filtrar tareas por categoría.
- Contador dinámico de tareas pendientes.
- Cambiar entre modo claro y modo oscuro.
- Guardar tareas y preferencias del usuario utilizando `localStorage`.

---

## Tecnologías utilizadas

- HTML5 para la estructura del contenido.
- CSS3 para el diseño visual, animaciones y modo oscuro.
- JavaScript para la lógica de la aplicación, manejo de eventos y persistencia de datos en el navegador.

---

## Persistencia de datos

Las tareas, el estado de completado y el tema seleccionado se almacenan en el navegador mediante `localStorage`, lo que permite conservar la información incluso después de recargar la página.
